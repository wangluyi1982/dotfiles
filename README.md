# Vim dot file

## Installation Guide:

```
git clone https://github.com/wangluyi1982/dotfiles .vim

cd ~/.vim

ln -s ~/.vim/vimrc ~/.vimrc
ln -s ~/.vim/louiswang.zsh-theme ~/.vim/oh-my-zsh/themes/
ln -s ~/.vim/oh-my-zsh ~/.oh-my-zsh
ln -s ~/.vim/zshrc ~/.zshrc

```

## set up neovim

ln -s ~/.vim/config/nvim ~/.config/
